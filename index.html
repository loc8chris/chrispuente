PROJECT DESCRIPTION

The objective of this project was to create a web application that served some meaningful purpose to the public. It was required to use ReactJS and or AngularJS in some way. It had to utilize a Node and Expesss Web Server. I inplemented a MongoDB database and Mongoose ORM. GET and POST routes were created to retrieve and add new data. I hard coded authentication of the user into the application. MVC folder structure was adhered to when creating the pathways.

I came up with an idea for an application that let's the user track their moods on a regular basis. The whole idea is to train the user to become more aware of their misguided thoughts and beliefs; to notice patterns in their feelings and how their feelings relate to the way they think. It's a thoughts and feelings journal of sorts.
The user chooses from an array of feeling options that describes how they feel at that very moment. Each feeling has three incomplete sentences that coincide with that emotion from which the user can choose to complete. The feeling and completed sentence are graded and color-coded depending on its level of bliss to rage. It's then stored in the user's profile within the database. Once enough data has been stored the user can access said data in any chosen span of time for it to be displayed on a color-coded chart to show patterns in feelings and behavior at any given moment. Hopefully they can use the app to become more aware and make some positive changes.
